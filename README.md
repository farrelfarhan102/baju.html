# Website EFRYA

Cara menjalankan:
1. Install Flask: `pip install flask`
2. Jalankan: `python app.py`
3. Buka browser: http://localhost:5000

Edit data brand, IG, WA, dan produk di bagian atas file `app.py` (dictionary BRAND dan list PRODUCTS).
Gambar produk ada di folder `static/images/` — kamu bisa ganti dengan foto produk lain kapan saja.
